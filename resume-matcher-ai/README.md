
# AI Resume Matcher using Endee Vector Database

This project demonstrates a resume-job matching system using vector embeddings and the Endee vector database.

## Features
- Resume similarity matching
- Vector search using embeddings
- AI-powered job recommendation

## Tech Stack
- Python
- Sentence Transformers
- Endee Vector Database
- Streamlit
